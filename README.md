# 🚲 Cyclistic Bike-Share Analysis (2025)

**How do annual members and casual riders use Cyclistic bikes differently?**

## 📌 Project Overview

This project answers Cyclistic's business question by analyzing **5.4 million bike rides** in Chicago during 2025.

**Goal:** Provide data-driven marketing recommendations to convert casual riders into annual members.

## 🛠️ Tools Used

| Tool | Purpose |
|------|---------|
| **Python (pandas)** | Data cleaning & analysis (5.4M rows) |
| **Tableau Public** | Interactive dashboard |
| **GitHub** | Project portfolio |

## 📊 Key Findings

| Metric | Casual | Member | Insight |
|--------|--------|--------|---------|
| **Total Rides** | 1.92M | 3.48M | Members ride **82% more** |
| **Avg Duration** | 19.9 min | 12.2 min | Casual rides **63% longer** |
| **Electric Bike** | 65% | 63% | Both prefer electric |
| **Peak Time** | Weekends, Afternoon | Weekdays, Morning | Different usage patterns |

## 💡 Recommendations

1. **"Summer Unlimited" Campaign** – Target casuals in May-June before peak season
2. **"Rush Hour" Offer** – Promote memberships during 4-6 PM (casual peak)
3. **"Electric First" Trial** – Free electric bike trial to convert casual users
4. **"Navy Pier" Activation** – On-site marketing at top casual stations
